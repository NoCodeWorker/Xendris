# Finitexo Code Matrix v0.4.1 - Expansion Intake

This package evaluates local expansion candidates for a future explicit
v0.4.x freeze.

It does not modify the v0.4 frozen dataset. It does not execute providers,
compare models, require network access, read `.env`, print secrets, or
authorize superiority/statistical claims.
