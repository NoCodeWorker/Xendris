# Finitexo Code Matrix v0.5

Provider smoke layer for the frozen Finitexo Code Matrix v0.4.3 n=10 dataset.

This phase validates execution plumbing, dataset loading, scoring, artifact
generation, costs, failures and conservative reporting.

Mock mode is the default. Real provider mode requires explicit configuration.

```txt
frozen_dataset_loaded != benchmark_claim_validated
provider_smoke_completed != provider_superiority_demonstrated
n10_smoke != statistical_significance
provider_failure != hidden_failure
scoring_summary != external_benchmark_validation
```
