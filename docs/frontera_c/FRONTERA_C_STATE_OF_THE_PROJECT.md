## 1. Executive Summary

Frontera C-Mayor is currently accepted as a bridge formal framework, not as validated theory and not as established new physics.

The accepted bridge result is:

```txt
D_CI(O) subset D_LC(O)
```

interpreted as a meaningful layered classification:

```txt
causal accessibility is necessary but not sufficient for causal-informational measurability, coherence access, and recoverability.
```

This framework is useful because it prevents causal access from being confused with measurable, coherent, recoverable information access. However, the current framework remains explainable by known physics: causal structure, information theory, measurement theory, decoherence, and recoverability.

## 3. What Frontera C-Mayor Is

Frontera C-Mayor is a disciplined bridge framework for modeling `c` as a causal-informational membrane:

```txt
a structure that separates domains by whether they can exchange coherent, causal, measurable, and recoverable information with an observer.
```

In its current accepted form, it is not a new law. It is a formal classification layer that organizes known constraints around observer-relative causal-informational access.

## 4. What Frontera C-Mayor Is Not

Frontera C-Mayor is not:

- validated,
- partially supported,
- established new physics,
- a replacement for relativity,
- a replacement for measurement theory,
- a replacement for information theory,
- a replacement for decoherence theory,
- an empirical benchmark program,
- an application layer,
- an auxiliary thermal, optical, visibility, contrast, camera, sensor, LiDAR, or `heating_power_W` study.

## 5. Formal Structure Accepted

### `D_LC(O)`

```txt
D_LC(O) = { e in E | A_c(O,e)=1 }
```

`D_LC(O)` is the domain of events causally accessible to observer `O` under `c`-constrained causal structure.

### `D_CI(O)`

```txt
D_CI(O) = { e in E |
  A_c(O,e)=1
  and I_c(O <- e)>0
  and M(O,e)=1
  and K(O,e)>=theta_K
  and R(O,e)>=theta_R
}
```

`D_CI(O)` is the domain of events that are causally accessible and also meet information-transfer, measurement, coherence-access, and recoverability requirements.

### `B_c(O)`

```txt
B_c(O) = boundary(D_CI(O), E \ D_CI(O))
```

`B_c(O)` is the candidate bridge boundary between causal-informational measurability and failure of at least one required layer.

It is not currently a proven physical boundary.

### `A_c(O,e)`

```txt
A_c(O,e)=1 iff e is causally accessible to O under c-constrained causal structure.
```

Classification:

```txt
KNOWN_PHYSICS
```

### `I_c(O <- e)`

```txt
I_c(O <- e)>0 iff a c-supported physical channel carries nonzero information from e to records available to O.
```

```txt
STANDARD_INFORMATION_THEORY / NEW_SYNTHESIS
```

### `M(O,e)`

```txt
M(O,e)=1 iff a physically allowed measurement relation can form an accessible record at O for the target event variable.
```

## 6. Key Accepted Claim

The key accepted bridge claim is:

```txt
Causal accessibility is necessary but not sufficient for causal-informational measurability.
```

Expanded:

```txt
An event may be causally accessible under c-constrained structure while failing information transfer, measurement possibility, coherence access, or recoverability.
```

This is not validation because:

- no theorem has created pressure beyond known physics,
- no empirical result directly pressures `c` as causal-informational membrane,
- no falsifiable core claim has passed,
- no external formal review has accepted non-redundancy,
- no auxiliary result is allowed to substitute for core validation.

The current status remains:

```txt
NOT_VALIDATED
```

## 8. Why This Is Not Established New Physics

The current bridge framework remains explainable by known physics:

- relativity already defines causal accessibility,
- information theory already defines channel capacity and information transfer,
- measurement theory already distinguishes causal reachability from record formation,
- decoherence already explains loss of coherence access,
- recoverability is already handled by inference, information, and reconstruction theory.

Therefore, novelty remains:

```txt
UNRESOLVED
```

## 9. Why It Remains Useful as Bridge Formal Framework

It remains useful because it separates layers that should not be collapsed:

```txt
causal access != information transfer != measurement != coherence access != recoverability
```

This helps prevent overclaiming, guides future formal work, and provides a disciplined map for deciding whether a future theorem exists or the framework should be demoted.

## 14. Conditions to Promote Back to CORE_THEORY_CANDIDATE

Promotion requires at least one of:

1. A theorem showing that `D_CI(O)` has nontrivial consequences not reducible to standard causal structure, measurement theory, information theory, decoherence, or recoverability.
2. A rigorous operational definition of `I_c`, `M`, `K`, `R`, `theta_K`, and `theta_R` that yields a falsifiable distinction.
3. A negative theorem clearly identifying where the framework collapses and where a non-collapsing residue remains.
4. A worked model where `B_c(O)` predicts or classifies something not already implied by known physics.
5. External formal review confirming that the classification is not merely vocabulary.

## 15. Conditions to Demote to INTERPRETIVE_FRAME_ONLY

Demotion is required if:

- `D_CI(O)` remains purely definitional,
- all examples are ordinary measurement or information limits,
- `theta_K` and `theta_R` remain arbitrary,
- `B_c(O)` never differs from standard observability boundaries,
- no falsifiable failure condition is produced,
- no nontrivial theorem or operational classification emerges.
