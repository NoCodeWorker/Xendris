# Mandatory Response Template — Frontera C Codex Work

Codex must use this template for all significant responses.

## Template

```markdown

- Current objective:
- Classification: CORE / BRIDGE / AUXILIARY / OFF-TRACK
- Link to Frontera C-Mayor:
- Drift risk:
- Allowed action:

## ACTION TAKEN

Describe what was done.

## OUTPUTS

List created or updated files.

## CLASSIFICATION

State whether each output is CORE, BRIDGE, AUXILIARY, or OFF-TRACK.

## VALIDATION IMPACT

Explain the impact separately:

- Impact on Frontera C-Mayor:
- Impact on bridge theory:
- Impact on auxiliary studies:
- Validation status change:

## DRIFT CHECK

Answer:

- Did this move away from `c` as causal-informational membrane?
- Did any auxiliary topic try to become core?
- Is a recenter action needed?

## NEXT ACTION

Recommend only one next action.
```

## Prohibited Response Pattern

Codex must not respond only with:

```text
Done.
Continued.
Implemented.
Next step is to improve benchmark.
```

without classification.

## Validation Language Control

Allowed:

```text
This improves the auxiliary model only.
This strengthens the conceptual bridge.
This does not validate Frontera C-Mayor.
```

Forbidden:

```text
This validates Frontera C.
```

unless the validation criteria in `08_PROJECT_STATUS_SCHEMA.md` are satisfied.

## Required Closing Line

Every major response must close with:

```text
Primary focus remains: c as causal-informational membrane.
```
