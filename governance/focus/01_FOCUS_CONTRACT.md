# Focus Contract — Frontera C-Mayor

## Contract ID

`FRONTERA_C_FOCUS_CONTRACT_V1`

## Non-Negotiable Objective

The project’s primary object is:

> **Frontera C-Mayor: the hypothesis that the speed of light `c` is not merely a kinematic propagation limit, but may operate as a causal-informational membrane separating physical domains according to their ability to exchange coherent, causal, measurable information with an observer.**

This objective cannot be replaced by any auxiliary experiment, benchmark, simulation, image-processing task, optical degradation study, or thermal-visibility model.

## Meaning of Frontera C

In this project, **Frontera C** refers to the role of `c` in the structure of physical reality, causal accessibility, information transfer, coherence, observability, and measurement.

It is about:

- the speed of light as causal boundary,
- light cones,
- relativistic causality,
- horizons,
- observer-dependent accessibility,
- information transfer,
- coherence and decoherence,
- measurable versus non-measurable domains,
- indirect traces of inaccessible physical domains.

## What Frontera C Is Not

Frontera C is not primarily:

- a camera problem,
- a thermal camera problem,
- a visibility problem,
- an image contrast problem,
- a LiDAR problem,
- a sensor-fusion problem,
- an atmospheric turbulence problem,
- a heat-haze problem,
- a computer vision problem,
- a `heating_power_W` benchmark,
- a commercial application framework.

These may exist only as auxiliary studies.

## Mandatory Classification

Every task, artifact, result, file, code path, experiment, benchmark, or theoretical note must be classified as one of:

### CORE

Directly belongs to Frontera C-Mayor.

Criteria:

- discusses `c`,
- relativistic causality,
- light cones,
- horizons,
- causal accessibility,
- observer-measurement relation,
- information transfer constrained by `c`,
- physical domains separated by causal-informational boundaries.

### BRIDGE

Connects the core hypothesis to nearby theoretical mechanisms.

Criteria:

- information theory,
- entropy,
- decoherence,
- coherence loss,
- measurement,
- recoverability,
- observer-relative physical access,
- indirect inference.

### AUXILIARY

Secondary experimental, computational, or analogical studies.

Criteria:

- visibility,
- contrast,
- image degradation,
- thermal turbulence,
- `heating_power_W`,
- cameras,
- optical propagation in media,
- signal corruption,
- local bench tests.

Auxiliary work may support a component, but never validates the main hypothesis by itself.

### OFF-TRACK

Does not help define, differentiate, formalize, test, falsify, or clarify Frontera C-Mayor.

Off-track work must be stopped or archived.

## Anti-Drift Rule

If a task begins with Frontera C-Mayor but moves toward an auxiliary topic without explicit justification, Codex must stop and emit:

```text
DRIFT DETECTED — RECENTERING REQUIRED
```

Then Codex must restate the primary objective and classify the attempted work.

## No Substitution Rule

Codex must never treat progress in an auxiliary study as validation of Frontera C-Mayor.

Correct:

```text
The thermal-optical benchmark advanced as an AUXILIARY study.
Frontera C-Mayor remains NOT_VALIDATED.
```

Incorrect:

```text
The contrast benchmark advanced, therefore Frontera C is close to validation.
```

## Contract Violation

The following actions violate this contract:

1. Continuing an auxiliary benchmark as if it were the main project.
2. Replacing `c` with contrast, visibility, heat, turbulence, or cameras.
3. Producing application lists unrelated to causal-informational membranes.
4. Claiming validation from analogies.
5. Failing to classify artifacts.
6. Failing to mark drift.
7. Optimizing code before the theoretical scope is restored.

## Primary Output Standard

Codex must prioritize:

1. strict definitions,
2. scope control,
3. alignment with known physics,
4. novelty separation,
5. falsifiability,
6. validation ladder,
7. artifact auditing,
8. controlled auxiliary registry.

## Final Clause

The project must never abandon this core without explicit human instruction:

> **`c` as causal-informational membrane.**
